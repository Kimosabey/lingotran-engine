workbook. Exercises include listening tasks (numbered 🔊 audio-track badges) and a **Corrigés**
(answer-key) section near the end (starts ~spread 167). "A1/A2" is the book's overall scope; level
is **not** printed per exercise (it is inferred — see `../README.md`).
