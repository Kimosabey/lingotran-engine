Les verbes *falloir, devoir* et *savoir*

**[A2 (inferred)]**
**115** Faites de phrases avec pouvoir et devoir comme dans l'exemple.

Exemple : Vous / écrire sans crayon / avoir un crayon.
→ Vous ne pouvez pas écrire sans crayon ; vous devez avoir un crayon.

a. Vous / partir sans visa / avoir un visa.
→ ___

→ ___

→ ___

→ ___

→ ___

→ ___
g. Ils / entrer sans le code / avoir le code.
→ ___

> **• Savoir**
> Le verbe savoir a 2 radicaux : *sai / sav*.
> **Je** sai**s.**     **Nous** sav**ons.**
> Il est suivi d'un infinitif ou d'une conjonction (*que, où, pourquoi, quand, si*, etc.).
> **Il** sait compter jusqu'à 10.     On **ne** sait **pas** si c'est vrai.

**[A2 (inferred)]**
**116** Complétez avec la bonne terminaison.

Exemple : On sait comment aller là-bas.

f. Vous sav ___ où est le musée d'Orsay ?
