### Les verbes *pouvoir, vouloir, devoir, falloir, savoir*

**• L'imparfait de l'indicatif des verbes *pouvoir, vouloir, devoir, falloir, savoir* —**

Pour ces verbes – à part *falloir* qui est particulier –, on suit la règle générale. L'imparfait se forme donc sur le radical du verbe à la 1<sup>re</sup> personne du pluriel du présent « nous » auquel on ajoute les terminaisons : *-ais / -ais / -ait / -ions / -iez / -aient*.

Pouvoir → nous pouv~~ons~~ → radical : *pouv*

Vouloir → nous voul~~ons~~ → radical : *voul*

Devoir → nous dev~~ons~~ → radical : *dev*

Savoir → nous sav~~ons~~ → radical : *sav*

Je **pouv**ais.    Je **voul**ais.    Je **dev**ais.    Je **sav**ais.

✋ Attention ! *Falloir* se conjugue seulement à la 3<sup>e</sup> personne du singulier.
L'imparfait est : Il **fallait**.
