INVENTARE

| temporal | | |
|---|---|---|
| | an + Dativ | **am** Morgen, **am** Dienstag |
| | bis + Akkusativ | **bis** bald, **bis** morgen |
| | in + Dativ | **im** Sommer, **in** den Ferien, **im** Mai |
| | nach + Dativ | **nach** dem Essen |
| | um + Akkusativ | **um** halb sieben, **um** 18.30 Uhr |
| | von … bis | **von** fünf **bis** sechs Uhr |
| | vor + Dativ | **vor** dem Konzert |
| | zu + Dativ | er kommt **zum** Frühstück |

| lokal | | |
|---|---|---|
| | an + Dativ / Akkusativ | Ferien **am** Meer, **an** den See fahren |
| | auf + Dativ | **auf** dem Marktplatz, **auf** dem Tisch |
| | aus + Dativ | **aus** Deutschland, **aus** der Schule |
| | bei + Dativ | **bei** Familie Bayer, **bei** Siemens |
| | in + Dativ / Akkusativ | **im** Park spielen, **in** die Stadt gehen |
| | nach + Dativ | **nach** Paris, **nach** Deutschland fahren |
| | von … nach + Dativ | **von** Hamburg **nach** Berlin |
| | zu + Akkusativ | **zur** Schule, **zu** meinem Freund gehen |

| modal | | |
|---|---|---|
| | für + Akkusativ | **für** meinen Freund, **für** dich |
| | mit + Dativ | **mit** dem Auto |
| | ohne + Akkusativ | **ohne** Hilfe, **ohne** dich |

Syntax

| **Verbzweitstellung** | | |
|---|---|---|
| | | Morgen **fahre** ich nach München. |
| | | Ich **fahre** morgen nach München. |

| **Verbergänzung** | | |
|---|---|---|
| | Nominativergänzung | Er heißt **Hans Bartels**. |
| | Akkusativergänzung | Ich nehme **einen Teller Suppe**. |
| | Dativergänzung | Gib **mir** bitte das Buch. |
| | Lokalergänzung | Wir wohnen **in der Goethestraße 17**. |
| | Qualitativergänzung | Der Film ist **langweilig**. |

| **Satzklammer** | | |
|---|---|---|
| | | Der Unterricht **fängt** um 9 Uhr **an**. |
| | | Ich **will** meine Tante **besuchen**. |
| | | **Komm** doch **mit**! |

| **Negation** | | |
|---|---|---|
| | *nicht* | Ich verstehe dich **nicht**. |
| | *kein* | Ich habe **keine** Ahnung. |

| **Fragesatz** | | |
|---|---|---|
| | Entscheidungsfrage | **Kommst** du heute? |
| | Ergänzungsfrage | **Wo** ist Peter? |

| **Satzverbindungen** | | |
|---|---|---|
| | *und* | Ich möchte eine Cola **und** Brot. |
