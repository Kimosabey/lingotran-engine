# Wiederholungsspiel

**1** „Drei in einer Reihe." Spielen Sie zu dritt.

[Spielanleitung-Kasten links:]

Jede/r Spieler/in braucht zehn Figuren: Zettel, Münzen, Spielfiguren, Bonbons …

[Fotos: eine Euro-Münze, ein eingewickeltes Bonbon, eine blaue Spielfigur]

Wer hat zuerst Geburtstag? Er/Sie beginnt.

Legen Sie eine Figur auf ein Feld. Lösen Sie die Aufgabe.

Richtig? → Die Figur bleibt auf dem Feld.

Falsch? → Die Figur muss weg.

Der/Die Nächste ist an der Reihe.

Wer hat zuerst drei Figuren in einer Reihe? Er/Sie gewinnt.

[Illustration: ein Spielfeld (Rechteck) mit farbigen Punkten (orange und rote Punkte verteilt).]

Niemand hat drei in einer Reihe? Spielen Sie noch einmal.
