# Wiederholungsspiel

**1** Spielen Sie zu fünft: zwei Spieler-Paare und ein Experte / eine Expertin.
Welches Spielerpaar hat am Ende die meisten Punkte?

## Spielregeln (linke Spalte)

Werfen Sie eine Münze:

Kopf 🪙
→ Spielen Sie einen Dialog zu dem Bild oben.

Zahl 🪙
→ Lösen Sie die Aufgabe unten.

Der Experte / Die Expertin entscheidet:
Wie war Ihr Dialog?

Sehr gut → 5 Punkte
Gut → 3 Punkte
Nicht so gut → 1 Punkt

War Ihre Antwort richtig? → 3 Punkte

Der Experte / Die Expertin notiert die Punkte. Er/Sie bekommt aus dem Lehrerhandbuch Informationen zu den Dialogen und Aufgaben.

## Spielbrett

### Start Team A (blaue Figur) → 1 → 2 → 3

Bilder oben (Team A): Buffet/Essen · Frau am Restauranttisch mit Kellner · Kalenderblatt <!-- picture label on calendar: "5. MAI" mit handschriftlicher Notiz, best-guess "Grill treffen?" -->

Feld 1:
Nennen Sie je ein Nomen für:
- Milchprodukte
- Obst
- Gemüse

Feld 2:
Wann haben Sie Geburtstag?

Feld 3 (Bild: Paar zeigt ein Foto):
Ergänzen Sie den Dialog:
○ Wer ist das?
● Das ist … Mutter.
  Und hier siehst du … Vater.

### Start Team B (grüne Figur) → 1 → 2 → 3

Bilder unten (Team B): Frau an der Supermarktkasse · Männer mit Kellner (Getränke auf Tablett) · Junge und Kind mit Fragezeichen

Feld 1:
Was kauft man wo? Nennen Sie je ein Nomen:
- in der Metzgerei
- im Supermarkt
- auf dem Markt

Feld 2:
Wann hat Ihr Freund / Ihre Freundin Geburtstag?

Feld 3 (Bild: Paar zeigt ein Foto):
Ergänzen Sie den Dialog:
○ Wer ist das?
● Das ist … Tochter.
  Und hier siehst du … Sohn.
