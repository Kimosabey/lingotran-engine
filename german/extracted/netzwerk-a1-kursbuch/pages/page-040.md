# Mit Buchstaben spielen

## 2 Mein Buchstabe. Ein Spieler / Eine Spielerin notiert auf einem Zettel einen Buchstaben. Die anderen nennen Wörter. Der Lehrer / Die Lehrerin schreibt die Wörter an die Tafel. Ist der Buchstabe im Wort? Der Spieler / Die Spielerin ruft „Ja!“ Ist der Buchstabe nicht im Wort? Der Spieler / Die Spielerin ruft „Nein!“ und der Lehrer / die Lehrerin streicht das Wort. Wer findet den Buchstaben?

[Bildergeschichte, Zeichnungen mit Sprechblasen:]
- *(Schüler nennen Wörter:)* Tag — Name — Flugzeug — zehn
- *(Spielerin antwortet:)* Ja! — Nein! — Ja! — Nein
- *(Tafel, handschriftlich:)* Tag / ~~Name~~ / Flugzeug / ~~zehn~~
- *(Schüler raten:)* A? — G?
- *(Spielerin zeigt Zettel mit „G“:)* G!
