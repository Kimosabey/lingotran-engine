# Sätze

## Aussagesätze  (K1, K5, K6, K10)

| Position 1 | Position 2 | | Satzende |
|------------|------------|--|----------|
| Niklas | wohnt | in Hamburg. | |
| Wir | können | nicht ins Kino | gehen. |
| Wir | holen | Sofia | ab. |
| Daniel | hat | sechs Stunden | gelernt. |
| Claudia | ist | zur Arbeit | gefahren. |

Im Aussagesatz steht das konjugierte Verb auf Position 2.

## Position im Satz  (K4)

| Position 1 | Position 2 | |
|------------|------------|--|
| **Lina** | isst | morgens Müsli. |
| Morgens | isst | **Lina** Müsli. |

Im Aussagesatz steht das Subjekt vor oder nach dem Verb.

## W-Fragen  (K1, K5, K6, K10, K12)

| Position 1 | Position 2 | | Satzende | Antwort |
|------------|------------|--|----------|---------|
| Wie | heißen | Sie? | | Ich heiße Oliver Hansen. |
| Welche Sprachen | sprichst | du? | | Spanisch und Deutsch. |
| Wen | hast | du zur Party | eingeladen? | Meine Freunde. |
| Wann | kannst | du | kommen? | Um acht. |
| Was | bringst | du | mit? | Ich bringe einen Kuchen mit. |

In der W-Frage steht das W-Wort auf Position 1. Das konjugierte Verb steht auf Position 2.
