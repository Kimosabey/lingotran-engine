# So war das

**4 a** 🔊 [2.39] Im Unicafé. Was hat Tina gemacht? Was hat Daniel gemacht? Hören Sie und kreuzen Sie an.

[Zeichnung: Daniel sitzt mit Laptop im Café, Tina steht mit Tasche daneben.]

| | Tina | Daniel |
|---|---|---|
| 1. für ein Seminar recherchieren | ☐ | ☐ |
| 2. Statistik lernen | ☐ | ☐ |
| 3. eine Prüfung machen | ☐ | ☐ |
| 4. eine Präsentation machen | ☐ | ☐ |
| 5. Fußball spielen | ☐ | ☐ |
| 6. im Studio trainieren | ☐ | ☐ |
| 7. in der Bäckerei arbeiten | ☐ | ☐ |

**b** Was haben die beiden gemacht? Schreiben Sie. Vergleichen Sie dann zu zweit.

| Tina | Daniel |
|---|---|
| hat für ein Seminar recherchiert | |

---

**G** (Grammatik) — **Partizip II: Verben auf** *-ieren*

Tina **hat** für ein Seminar **recherchiert**.

- recherchieren → hat recherchier**t**
- *ebenso:* telefon**ieren**, stud**ieren**, train**ieren**

**5 a** 🔊 [2.40] Was hat Daniel vor dem Studium gemacht? Hören Sie und ordnen Sie zu.

| # | Satzanfang | | Satzende |
|---|---|---|---|
| 1. | Daniel hat nach der Schule _D_ | A | seine Freunde kaum gesehen. |
| 2. | Bei der Arbeit hat es ___ | B | seine Freundin Maya getroffen. |
| 3. | Daniel hatte nie Zeit und hat ___ | C | auch Probleme gegeben. |
| 4. | Dann hat Daniel ___ | D | Arbeit bei einer Bank gefunden. |
| 5. | Im Urlaub hat Daniel ___ | E | vier Wochen Urlaub genommen. |

**b** Markieren Sie die Partizip-Formen in 5a. Machen Sie dann eine Liste mit dem Infinitiv und dem Partizip II.

_unregelmäßige Verben_
_finden – hat gefunden_

---

**G** (Grammatik) — **Partizip II: unregelmäßige Verben**

- finden → hat ge**fund**en
- treffen → hat ge**troff**en

**❗** Lernen Sie Infinitiv und Partizip II immer zusammen. Eine Liste mit unregelmäßigen Verben finden Sie im Anhang.
