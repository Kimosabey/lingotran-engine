# Alles fertig

**6 a** **Die Einweihungsfeier. Sehen Sie das Bild an. Was ist wo?**

> *Der Kühlschrank steht im Flur.*

> *Das Sofa ist im …*

**G**

**Wo? ⊙ *in* + Dativ**
**Wo** steht/ist der Schrank?
der Flur → (**In dem**) **Im** Flur.
das Bad → (**In dem**) **Im** Bad.
die Küche → **In der** Küche.
**Wo** sind die Bücher?
die Regale → **In den** Regalen.

**G**

**Wo? ⊙ *an, auf, hinter, in, neben, über, unter, vor, zwischen* + Dativ**
**Wo** ist das Bild?
**Über dem** Fernseher.

[▶ G3] [📖]
**b** **Wo genau sind die Dinge? Erzählen Sie.**

Präpositionen-Diagramme (Haus-Symbol mit rotem Punkt zur Positionsangabe):

| Präposition | Position des Punktes |
|---|---|
| an | am rechten Rand des Hauses |
| auf | oben auf dem Dach |
| hinter | hinter dem Haus (rechts, teilweise verdeckt) |
| in | im Haus |
| neben | neben dem Haus (rechts) |
| über | über dem Haus |
| unter | unter dem Haus |
| vor | vor dem Haus (unten) |
| zwischen | zwischen zwei Häusern |

> *Der Teppich ist im Wohnzimmer unter dem Sofa.*
