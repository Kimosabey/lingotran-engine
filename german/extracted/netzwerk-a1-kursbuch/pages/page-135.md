## Redemittel

### über Kleidung sprechen

- ○ Sieh mal, der Mantel ist doch toll, oder?
- ● Welcher denn?
- ○ Na, dieser hier.
- ● Findest du? Also, ich finde diese Jacke hier viel besser.
- ● Ja, genau.
- ○ Welche Jacke meinst du? Diese?
- ○ Oh ja, gut, dann nehme ich die Jacke.

### Gespräche beim Kleiderkauf führen

| Verkäufer | Kunde |
|---|---|
| Kann ich Ihnen helfen? | Ich suche einen Pullover. |
| Wie gefällt Ihnen dieser Pullover? | Sehr gut. / Nicht so gut. |
| Welche Größe haben/brauchen Sie? | Ich glaube, L oder XL. |
| Passt Ihnen der Pullover? | Nein, er ist zu klein/groß/eng/weit/kurz/lang. Haben Sie ihn auch in L? |
| Diese Jacke passt/steht Ihnen gut. | Ich weiß nicht. Grün gefällt mir nicht (so gut). |

### sich im Kaufhaus orientieren

| (Kunde) | (Verkäufer) |
|---|---|
| Entschuldigung, wo finde/kriege ich …? / Wo gibt es …? | Das gibt es im dritten/vierten/fünften/… Stock. / … finden Sie im Erdgeschoss/Untergeschoss. |
| Ich suche … / Ich brauche … / Haben Sie …? | Tut mir leid, das haben wir nicht. |
| Ich danke Ihnen. / Danke für Ihre Hilfe. | |

## Grammatik

### Interrogativartikel: *Welcher? Welches? Welche?*

| | Nominativ | Akkusativ | Dativ |
|---|---|---|---|
| der Mantel | Welch**er**? | Welch**en**? | Welch**em**? |
| das Kleid | Welch**es**? | Welch**es**? | Welch**em**? |
| die Jacke | Welch**e**? | Welch**e**? | Welch**er**? |
| die Schuhe | Welch**e**? | Welch**e**? | Welch**en**? |

### Demonstrativartikel: *dieser, dieses, diese*

| Nominativ | Akkusativ | Dativ |
|---|---|---|
| dies**er** | dies**en** | dies**em** |
| dies**es** | dies**es** | dies**em** |
| dies**e** | dies**e** | dies**er** |
| dies**e** | dies**e** | dies**en** |

### Verben mit Dativ

| Verb | Beispiel |
|---|---|
| gefallen | Der Pullover **gefällt mir** sehr gut. |
| stehen | Die Schuhe **stehen ihm** nicht. |
| passen | Die Jacke **passt ihr** nicht. |
| helfen | Kann ich **Ihnen helfen**? |

### Personalpronomen im Dativ

| | | | |
|---|---|---|---|
| ich | **mir** | wir | **uns** |
| du | **dir** | ihr | **euch** |
| er | **ihm** | sie | **ihnen** |
| es | **ihm** | | |
| sie | **ihr** | Sie | **Ihnen** |

### Partizip II: Verben mit Präfix

**trennbare Verben**

| Infinitiv | Partizip II |
|---|---|
| ankommen | ist an**ge**kommen |
| umtauschen | hat um**ge**tauscht |
| anziehen | hat an**ge**zogen |

**nicht trennbare Verben**

| Infinitiv | Partizip II |
|---|---|
| bezahlen | hat bezahlt |
| empfehlen | hat empfohlen |
| erzählen | hat erzählt |

trennbare Präfixe:
*ab-, an-, auf-, aus-, ein-, mit-, zu-, zurück- …*

nicht trennbare Präfixe:
*be-, emp(f)-, ent-, er-, ge-, ver-, zer-*
