## die Uhrzeit nennen

| Frage | inoffiziell | 🕑 14:45 offiziell |
| --- | --- | --- |
| Wie spät ist es? / Wie viel Uhr ist es? | Es ist Viertel vor drei. / Es ist halb zwei. | Es ist vierzehn Uhr fünfundvierzig. / Es ist dreizehn Uhr dreißig. |
| Wann? | Um zehn nach neun. / Um kurz vor eins. | Um neun Uhr zehn. / Um zwölf Uhr achtundfünfzig. |

## einen Termin vereinbaren

| | |
| --- | --- |
| Ich hätte gern einen Termin. / Haben Sie heute/morgen / am … einen Termin? | Ja. Da geht es um 14:15 Uhr. / Nein, heute/morgen / am … geht es (leider) nicht, aber am … |
| Können Sie am … um … kommen? / Geht es am … um … Uhr? | Ja, da kann ich. / Nein, da kann ich leider nicht. / Ja, das geht. / Nein, das geht leider nicht. |

## sich für eine Verspätung entschuldigen … / … und darauf reagieren

| sich für eine Verspätung entschuldigen … | … und darauf reagieren |
| --- | --- |
| Entschuldigung, bitte. | Schon gut. |
| Bitte entschuldigen Sie. | Kein Problem. |
| Ich bitte um Entschuldigung. | Macht nichts. |
| Es tut mir leid, ich bin zu spät. | Das nächste Mal bitte pünktlich! |

## Zeitangaben: *am, um, von … bis*

| | Wochentage/Tageszeiten | Uhrzeit |
| --- | --- | --- |
| Wann? | **am** Montag / **am** Vormittag | **um** Viertel vor drei |
| Wie lange? | **von** Montag **bis** Samstag | **von** neun **bis** halb zwei |
