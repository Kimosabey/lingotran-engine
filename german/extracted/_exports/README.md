# German A1 — Extraction Deliverables

Clean, content-team-ready exports. **All CSVs are UTF-8 with BOM** so German
umlauts (ä ö ü ß) render correctly on double-click in Excel / Google Sheets.

## How this is organised

- `_combined/` — roll-up sheets: **global** `german-a1-*` (every book) and
  **per-publisher** `goethe-a1-*` / `netzwerk-a1-*` / `goyal-a1-*`.
- `<publisher>/<book>/` — the four deliverables for one book:
  the unified `.md`, and `-catalog` / `-questions` / `-vocabulary` CSVs
  (a book only has the sheets that apply to it).

## Sheet columns

- **catalog** — one row per page: section, content type, activity, topic, level, status, word count, summary.
- **questions** — one row per item: section, teil, item, item_type, question, option_a/b/c, correct_answer, topic, source_page.
- **vocabulary** — one row per word: word, article, plural, word_class, example, topic, source_page.
