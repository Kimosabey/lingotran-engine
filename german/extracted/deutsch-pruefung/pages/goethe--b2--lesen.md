### About the Section

![Study bear mascot](https://deutsch-pruefung.de/_next/image?url=%2Fsection-assets%2Fbear-mascot.webp&w=384&q=75)

The Goethe B2 Lesen (Reading) module takes 65 minutes. You read multiple texts and answer questions. You can start with any task. Each task has only one correct answer. Only answers on the answer sheet are graded. No aids (dictionaries, phones, mobile devices) are allowed.

#### Teil 1: Einstellungen/Haltungen verstehen (Understanding Attitudes)

• Match texts to statements about attitudes or opinions

• Task type: Zuordnung (Matching) - 4 texts

#### Teil 2: Informationen verstehen (Understanding Information)

• Read texts and understand detailed information

• Task type: Zuordnung (Matching) - 6 texts

#### Teil 3: Informationen verstehen (Understanding Information)

• Read texts and answer comprehension questions

• Task type: Wahl (3-gliedrig MC) - Multiple Choice with 3 options

#### Teil 4: Standpunkte verstehen (Understanding Viewpoints)

• Read texts and identify different viewpoints

• Task type: Zuordnung (Matching) - 8 texts

#### Teil 5: Regeln/Instruktionen verstehen (Understanding Rules/Instructions)

• Read formal texts with rules or instructions

• Task type: Zuordnung (Matching) - 3 paragraphs
