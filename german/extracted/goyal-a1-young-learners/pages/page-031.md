## B. Sätze bauen

Schneide die Kärtchen aus, mische sie und bau damit unterschiedliche Sätze.

Kärtchen-Gruppe 1 (Fragewörter):

| | | | |
|---|---|---|---|
| Wo | Wie | Wer | Wie alt |

Kärtchen-Gruppe 2 (Verben):

| | | | |
|---|---|---|---|
| lieg-t | wohn-st | wohn-t | wohn-en |
| heiß-t | heiß-en | bist | ist |
| sind | | | |

Kärtchen-Gruppe 3:

| | | | |
|---|---|---|---|
| Augsburg? | München? | die Weigels? | deine Eltern? |
| Herr Scholz? | wir? | du? | deine Tante? |
| Martina? | in Berlin? | in Köln? | deine Adresse? |
