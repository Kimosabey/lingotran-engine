# German · audio (raw audio/listening sources)

Raw, officially-published listening-companion files for the German corpus —
downloaded manually, not scraped. The Goethe-Institut serves these as MP4
containers (audio track only); keep the `.mp4` extension as served.

Not tracked in git (see root `.gitignore`) — structure only.

## Layout

```
audio/
  <publisher>/
    <exam-variant>/
      <file>.mp4
```
