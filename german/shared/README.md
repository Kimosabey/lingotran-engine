# German · shared

Code shared across German source-acquisition channels (`web/`, and future
`pdf/`, `ocr/`, `audio/`, …) — e.g. corpus-path helpers, slug/frontmatter
utilities, and `manifest.tsv` read/write shared with all channels.

Empty by design in this phase. Promote a helper here only once a **second**
channel needs it — until then it stays inside `web/` to avoid premature
abstraction. Nothing German-web-specific belongs here.
