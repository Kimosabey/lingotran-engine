Visual Brand Guidelines

Logo, colour, type, and components. Built on Nunito.

01 · Logo

The wordmark.

Three colourways. Use the one that gives cleanest contrast on its background.

Primary · Brand Purple on light
Reversed · White on purple / dark
Monochrome · Black on light

App icon

Colour and mono have transparent backgrounds for light tiles. The reversed icon ships white-on-black.

Colour
Mono black
Reversed white
Clear space: a margin equal to the height of the cap-and-bubble mark on all sides. Minimum width 120px for the wordmark, 32px for the icon. The supplied colour wordmark file ships on a solid black background; for light layouts use the transparent purple lockup included alongside this document.

Misuse

Don't recolour outside purple, black, or white.
Don't place the purple mark on coral or busy backgrounds. Use white reversed on dark.
Don't stretch, rotate, skew, or add shadows.
Don't crowd the mark or drop below minimum size.

05 · Text blocks

Setting body copy.

Keep a comfortable measure. Open with a lead line in 700, then body in 500.

A lead paragraph sits at 21px / 700 and frames the block.

Body paragraphs run at 18px / 500 with 30px line-height and a max width near 640px, which keeps lines around 70 to 80 characters for easy reading. Paragraphs are separated by 24px of space, never by indents.

Use a single emphasis colour per block. Put weight on the word that matters with an inline purple or, more sparingly, an inline coral highlight. Do not underline for emphasis and do not mix both colours in one sentence.

<div style="max-width:640px;">
  <p style="font:700 21px/1.55 Nunito;color:#1A1A2E;margin:0 0 22px;">Lead line.</p>
  <p style="font:500 18px/1.75 Nunito;color:#444;margin:0 0 24px;">
    Body copy with an <span style="color:#41009A;font-weight:700;">inline highlight</span>.
  </p>
</div>
06 · Callouts

Calling something out.

In a heading

One emphasised word inside an H1 or H2. Coral, italic, same size and 800 weight. One word, never a phrase.

Built for institutions.
<h2>Built for <span style="color:#FF6B4A;font-style:italic;font-weight:800;">institutions.</span></h2>

In text — inline highlight

Within a paragraph, colour the key word in brand purple 700 (or coral 700 for warmth). No background, no underline.

Every student practises between classes, so you walk in knowing exactly where each one stands.

<span style="color:#41009A;font-weight:700;">key phrase</span>

In text — callout box

For a pulled-out point or tip inside body copy. Lilac fill, brand-purple left rule, optional uppercase label.

Key point

A callout box separates a single important takeaway from the surrounding paragraphs without shouting.

<div style="background:#F0E6FF;border-left:4px solid #41009A;border-radius:0 12px 12px 0;padding:16px 20px;">
  <span style="font:700 11px Nunito;letter-spacing:1px;text-transform:uppercase;color:#41009A;">Key point</span>
  <p style="margin:6px 0 0;color:#412485;font:600 16px/1.6 Nunito;">Your takeaway.</p>
</div>

Pill badge on light, plain label on dark. Both sit above a heading.

10 · Quick reference

Tokens.

Paste into a prompt or stylesheet to keep generated visuals on-brand.

:root{
  --brand-purple:#41009A;  --violet:#7C4DFF;  --pill-purple:#412485;
  --coral:#FF6B4A;  --success:#00B894;  --alert:#FF4D4D;
  --ink:#1A1A2E;  --ink-deep:#1A0B2E;  --body:#444444;  --muted:#666666;
  --border:#DDDDDD;  --caption:#AAAAAA;
  --lilac:#F0E6FF;  --cream:#FAF6F0;  --off-white:#FAFAFA;  --white:#FFFFFF;
  --font:'Nunito','Helvetica Neue',Helvetica,Arial,sans-serif;
}
/* Scale: H1 800/56/64 · H2 800/46/56 · H3 700/42 · H4 800/26 · H5 700/20
   Lead 700/21 · Body 500/18, line-height 1.75, 24px between paragraphs · Caption 400/13/20 · measure 640px
   Callout-in-heading: coral italic 800 · Inline highlight: brand-purple 700
   Callout box: lilac + #41009A left rule · Pill: 700/11 upper, pill-purple on lilac
   Stat: 800/44, coral on dark / brand-purple on light / green for positive
   Button: 600/16 white on brand-purple, radius 8 · Brand gradient 135deg #41009A → #7C4DFF */

For email, cap H1 near 30 to 32px, use px line-heights, inline all CSS, and treat the Nunito link as progressive enhancement (Outlook falls back to Arial).
