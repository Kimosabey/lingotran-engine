Overview

What this engine does

A scalable workflow for extracting structured linguistic data from scanned language-learning PDFs — converting textbook content into clean, faithful material for study tools.

The source PDFs are scanned images with no text layer, so content is recovered by Claude-vision transcription of 300-DPI page renders. The guiding principle is zero data loss: every heading, instruction, table, exercise, blank, example, answer key, caption and page number is captured verbatim, then independently verified and repaired until faithful.

French is the first fully built-out corpus. The same pipeline is designed to extend to German, Japanese, Portuguese, Romanian, Russian and Spanish.

Faithful, not interpreted. The only non-verbatim addition is an inferred CEFR level tag per exercise — always marked “(inferred)”.

Pipeline

The extraction workflow

One workflow — transcribe-french-pages — runs each page through three stages. Every agent call runs at effort: high.

Resumable by design. Each stage writes its result to disk (a Markdown page and a JSON verdict), so a multi-session job can stop and resume from manifest.tsv at any point.
